# Queries

## `balance`

Returns the <a href="/docs/reference/objects/#balance">`Balance!`</a> of a specific address for a given asset id.

**args:**

`owner`: <a href="/docs/reference/scalars/#address">`Address!`</a>

The owner address.

`assetId`: <a href="/docs/reference/scalars/#assetid">`AssetId!`</a>

The asset id.

## `balances`

Returns a <a href="/docs/reference/objects/#balance">`BalanceConnection!`</a> for an array of balances for each asset owned by a given address.

**args:**

`filter`: <a href="/docs/reference/objects/#balancefilterinput">`BalanceFilterInput!`</a>

A filter to specify the wallet owner address.

## `block`

Returns information about a certain <a href="/docs/reference/objects/#block">`Block`</a>. Accepts either the block id or block height as an argument.

**args:**

`id`: <a href="/docs/reference/scalars/#blockid">`BlockId`</a>

The block id.

`height`: <a href="/docs/reference/scalars/#u64">`U64`</a>

The block height.

## `blocks`

Returns a <a href="/docs/reference/objects/#block">`BlockConnection!`</a> for an array of all blocks produced.

## `chain`

Returns <a href="/docs/reference/objects/#chaininfo">`ChainInfo!`</a> about the target Fuel network used for the API.

## `coin`

Returns details about a specific <a href="/docs/reference/objects/#coin">`Coin`</a>.

**args:**

`utxoId`: <a href="/docs/reference/scalars/#utxoid">`UtxoId!`</a>

A unique 32 byte identifier for the UTXO.

## `coins`

Returns a <a href="/docs/reference/objects/#coin">`CoinConnection!`</a> for an array of coins based on a given owner and/or asset id

**args:**

`filter`: <a href="/docs/reference/objects/#coinfilterinput">`CoinFilterInput!`</a>

A filter with the owner address and optionally the asset id.

## `contract`

Returns the <a href="/docs/reference/objects/#contract">`Contract`</a> information for a given contract id.

**args:**

`id`: <a href="/docs/reference/scalars/#contractid">`ContractId!`</a>

The contract id of the requested contract.

## `contractBalance`

Returns the <a href="/docs/reference/objects/#contractbalance">`ContractBalance!`</a> for a given contract and asset id.

**args:**

`contract`: <a href="/docs/reference/scalars/#contractid">`ContractId!`</a>

The contract that owns the balance.

`asset`: <a href="/docs/reference/scalars/#assetid">`AssetId!`</a>

The asset id for the balance.

## `contractBalances`

Returns a  <a href="/docs/reference/objects/#contractbalance">`ContractBalanceConnection!`</a> for an arrray of balances for all assets owned by a given contract

**args:**

`filter`: <a href="/docs/reference/objects/#contractbalancefilterinput">`ContractBalanceFilterInput!`</a>

A filter for the contract balances.

## `health`

Returns `true` if the API is running or `false` if the API is down.

## `messageProof`

Returns the <a href="/docs/reference/objects/#messageproof">`MessageProof`</a> for a given message id or transaction.

**args:**

`transactionId`: <a href="/docs/reference/scalars/#transactionid">`TransactionId!`</a>

The transaction id for the message.

`messageId`: <a href="/docs/reference/scalars/#messageid">`MessageId!`</a>

The id of the message.

## `messages`

Returns a <a href="/docs/reference/objects/#message">`MessageConnection!`</a> for an array of messages for a given owner.

**args:**

`owner`: <a href="/docs/reference/scalars/#address">`Address`</a>

The owner address of the messages.

## `nodeInfo`

Returns <a href="/docs/reference/objects/#nodeinfo">`NodeInfo!`</a> about the current node.

## `resourcesToSpend`

Returns the list of spendable <a href="/docs/reference/unions/#resource">`[[Resource!]!]!`</a> per asset from the query. The length of the result is the same as the length of `query_per_asset`. The ordering of assets and `query_per_asset` is the same.

**args:**

`owner`: <a href="/docs/reference/scalars/#address">`Address!`</a>

The address to send coins to.

`queryPerAsset`: <a href="/docs/reference/objects/#spendqueryelementinput">`[SpendQueryElementInput!]!`</a>

The list of requested asset resources. Several entries with the same asset id are not allowed.

`excludedIds`: <a href="/docs/reference/objects/#excludeinput">`ExcludeInput`</a>

The resources to exclude.

## `transaction`

Returns <a href="/docs/reference/objects/#transaction">`Transaction`</a> details for a given transaction id.

**args:**

`id`: <a href="/docs/reference/scalars/#transactionid">`TransactionId!`</a>

The id for the transaction.

## `transactions`

Returns a <a href="/docs/reference/objects/#transaction">`TransactionConnection!`</a> for an array of all transactions.

## `transactionsByOwner`

Returns a <a href="/docs/reference/objects/#transaction">`TransactionConnection!`</a> for an array of all transactions from a given address.

**args:**

`owner`: <a href="/docs/reference/scalars/#address">`Address!`</a>

The owner address of the transactions.
