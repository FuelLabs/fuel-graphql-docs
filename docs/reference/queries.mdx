# Queries

## `balance(...)`: <a href="/docs/reference/objects/#balance">`Balance!`</a>

Returns the balance of a specific address for a given asset id.

**args:**

`owner`: <a href="/docs/reference/scalars/#address">`Address!`</a>

The owner address.

`assetId`: <a href="/docs/reference/scalars/#assetid">`AssetId!`</a>

The asset id.

## `balances(...)`: <a href="/docs/reference/objects/#balance">`BalanceConnection!`</a>

Returns a connection for an array of balances for each asset owned by a given address.

**args:**

`filter`: <a href="/docs/reference/objects/#balancefilterinput">`BalanceFilterInput!`</a>

A filter to specify the wallet owner address.

## `block(...)`: <a href="/docs/reference/objects/#block">`Block`</a>


Returns information about a certain block. Accepts either the block id or block height as an argument.

**args:**

`id`: <a href="/docs/reference/scalars/#blockid">`BlockId`</a>

The block id.

`height`: <a href="/docs/reference/scalars/#u64">`U64`</a>

The block height.

## `blocks(...)`: <a href="/docs/reference/objects/#block">`BlockConnection!`</a>

Returns a connection for an array of all blocks produced.

## `chain`: <a href="/docs/reference/objects/#chaininfo">`ChainInfo!`</a>

Returns information about the target Fuel network used for the API.

## `coin(...)`: <a href="/docs/reference/objects/#coin">`Coin`</a>

Returns details about a specific coin.

**args:**

`utxoId`: <a href="/docs/reference/scalars/#utxoid">`UtxoId!`</a>

A unique 32 byte identifier for the UTXO.

## `coins(...)`: <a href="/docs/reference/objects/#coin">`CoinConnection!`</a>

Returns a connection for an array of coins based on a given owner and/or asset id

**args:**

`filter`: <a href="/docs/reference/objects/#coinfilterinput">`CoinFilterInput!`</a>

A filter with the owner address and optionally the asset id.

## `contract(...)`: <a href="/docs/reference/objects/#contract">`Contract`</a>

Returns the contract information for a given contract id.

**args:**

`id`: <a href="/docs/reference/scalars/#contractid">`ContractId!`</a>

The contract id of the requested contract.

## `contractBalance(...)`: <a href="/docs/reference/objects/#contractbalance">`ContractBalance!`</a>

Returns the balance for a given contract and asset id.

**args:**

`contract`: <a href="/docs/reference/scalars/#contractid">`ContractId!`</a>

The contract that owns the balance.

`asset`: <a href="/docs/reference/scalars/#assetid">`AssetId!`</a>

The asset id for the balance.

## `contractBalances(...)`: <a href="/docs/reference/objects/#contractbalance">`ContractBalanceConnection!`</a>

Returns a connection for an arrray of balances for all assets owned by a given contract

**args:**

`filter`: <a href="/docs/reference/objects/#contractbalancefilterinput">`ContractBalanceFilterInput!`</a>

A filter for the contract balances.

## `health: Boolean!`

Returns `true` if the API is running or `false` if the API is down.

## `messageProof(...)`: <a href="/docs/reference/objects/#messageproof">`MessageProof`</a>

Returns the message proof for a given message id or transaction. Only available for the beta-2 testnet.

**args:**

`transactionId`: <a href="/docs/reference/scalars/#transactionid">`TransactionId!`</a>

The transaction id for the message.

`messageId`: <a href="/docs/reference/scalars/#messageid">`MessageId!`</a>

The id of the message.

## `messages(...)`: <a href="/docs/reference/objects/#message">`MessageConnection!`</a>

Returns a connection for an array of messages for a given owner.

**args:**

`owner`: <a href="/docs/reference/scalars/#address">`Address`</a>

The owner address of the messages.

## `nodeInfo`: <a href="/docs/reference/objects/#nodeinfo">`NodeInfo!`</a>

Returns information about the current node.

## `resourcesToSpend(...)`: <a href="/docs/reference/unions/#resource">`[[Resource!]!]!`</a>

Returns the list of spendable resources per asset from the query. Only available for the beta-2 testnet. The length of the result is the same as the length of `query_per_asset`. The ordering of assets and `query_per_asset` is the same.

**args:**

`owner`: <a href="/docs/reference/scalars/#address">`Address!`</a>

The address to send coins to.

`queryPerAsset`: <a href="/docs/reference/objects/#spendqueryelementinput">`[SpendQueryElementInput!]!`</a>

The list of requested asset resources. Several entries with the same asset id are not allowed.

`excludedIds`: <a href="/docs/reference/objects/#excludeinput">`ExcludeInput`</a>

The resources to exclude.

## `transaction(...)`: <a href="/docs/reference/objects/#transaction">`Transaction`</a>

Returns transaction details for a given transaction id.

**args:**

`id`: <a href="/docs/reference/scalars/#transactionid">`TransactionId!`</a>

The id for the transaction.

## `transactions(...)`: <a href="/docs/reference/objects/#transaction">`TransactionConnection!`</a>

Returns a connection for an array of all transactions.

## `transactionsByOwner(...)`: <a href="/docs/reference/objects/#transaction">`TransactionConnection!`</a>

Returns a connection for an array of all transactions from a given address.

**args:**

`owner`: <a href="/docs/reference/scalars/#address">`Address!`</a>

The owner address of the transactions.
